下拉选择可用于在多个选项之间进行选择，可以选择一个或多个选项。组件显示当前状态和向下箭头。

md-select 组件可兼容原生 select 标签的各种属性，包括 required 和 multiple 等。

下拉选择

md-select 需要和 md-option 组件搭配使用，后者提供可能的选择项，且必须指定 value 属性。同时，md-select 组件应该在 md-field 组件内使用。

如果向 md-select 传递的 value 属性是数组，则代表使用多选功能。
