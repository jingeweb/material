按钮组件用于响应用户的点（触）击操作。按钮类型包括常规按钮、图标按钮、分组按钮和 FAB 按钮。请参看 Material Design 3 设计规范手册 以获取更多指导。

常规按钮

常规按钮适用于各类交互。

API - md-button

除click外，任何原生 DOM 事件都可以通过组件绑定和使用。

图标按钮

除了常规按钮的 4 种类型，图标按钮还支持切换选中状态。

API - md-icon-button

除click外，任何原生 DOM 事件都可以通过组件绑定和使用。

分组按钮

分组按钮的功能和 Tab 类似。

API - md-segmented-button

API - md-segmented-button-item

FAB 按钮

每屏最多只使用一个 FAB 按钮，并且仅在按钮的目的是触发屏幕的主要操作时才使用。否则，请使用 md-icon-button。

API - md-fab
