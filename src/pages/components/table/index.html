表格组件用于显示原始数据集，通常被用在桌面（而不是移动平台）产品中。表格组件支持许多属性来更改和影响布局。

虽然 Jinge Material 项目，但表格组件的接口和使用的设计，参考的是 Element 组件库。

基础表格

表格由 md-table 和 md-table-column 两个组件构成。前者用于创建表格，后者用于在表格内定义每一列。

可选模式

表格可开启可选择模式，该模式下可以通过 Checkbox 来选择行数据。

自定义模版

可以通过为 md-table-column 指定 slot 来使用自定义模版。 其中，传送默认的 slot，可指定表格内容(td)模版；传递 head slot，可指定表头(th)的模版。

表头 slot 用于渲染每一个表头，可使用的数据包括 column 和 columnIndex。 表格内容 slot 用于渲染每一行的每一个单元格，可使用的数据包括：

  • data: 单元格对应的行数据。
  • index: 单元格对应的行的索引。
  • selected: 单元格对应的行是否被选中。只在表格开启可选模式下有用。
  • column: 单元格对应的列。
  • columnIndex: 单元格对应的列的索引。

行的自定义样式

如果表格的某一行需要有特别的样式，可以通过 rowClass 属性为该行指定额外的样式类。 这个属性必须指定为函数，函数接收行数据，返回样式类。
