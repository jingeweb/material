Material Design 中的响应式布局可适应任何可能的屏幕尺寸，该布局方案包括一个灵活的网格系统，以确保各个布局之间的一致性。

断点

使用断点来控制响应式布局。断点可用于 md-size-[breakpoint]-[amount]、md-hide-[breakpoint] 这两类 class。

布局项大小

默认情况下布局项等分容器。可以使用 md-size-[amount] 可以指定元素在布局中的大小，例如 md-size-30。其中，[amount] 的值可以是 33、66 或 5 的倍数。

使用 md-[breakpoint]-size-[amount] 可以基于断点来更改布局大小。

隐藏元素

可以使用 md-[breakpoint]-hide 类来实现在较小屏幕上隐藏某个元素。 breakpoint 可以是上述的任意断点。
